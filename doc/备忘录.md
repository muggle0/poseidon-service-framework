# 基础的设计模式

组件式的开发方式，各个组件各司其职。

common 存放基类，公共接口用以各个组件之间的交互

core 模块存放配置类 需要共同使用的类。core 模块只能依赖common

# api 网关所提供的功能

提供 api 列表，sso,

## 开发日志
